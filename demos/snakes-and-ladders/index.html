Snakes and Ladders
Dana Y.

The clues below each ladder provide words (in alphabetical order) for that ladder. One piece of each word ladder is provided in the grid. Enter words such that only one letter changes per rung.

Also, solve a slitherlink (draw a loop) treating all the cells as one grid. The numbers indicate how many of that cell’s sides are part of the loop. For more information about solving this type of puzzle: https://www.puzzle-loop.com/faq.php, visit www.puzzle-loop.com/faq.php

To get the answer, for each row take the letter of the leftmost cell in that row that is both inside the loop and touches the loop on three sides. Some rows will not have any such cell.
